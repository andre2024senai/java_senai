# Java Fundamentos – SENAI

Este repositório contém os materiais, exemplos e exercícios utilizados nas aulas de **Java – Fundamentos**.

## Organização do conteúdo
- Os materiais estão organizados por **módulos**
- Cada módulo possui exemplos práticos em Java
- Utilize este repositório como **material de apoio às aulas**

## Orientações
- Leia os arquivos `README.md` de cada pasta
- Analise os exemplos de código
- Execute os códigos na IDE

## Documentação do JAVA

### Você pode acessar a documentação oficial do Java em:
https://docs.oracle.com/en/java/index.html

De lá, os principais caminhos são:

### Para o java.util (Scanner, ArrayList, etc.):
https://docs.oracle.com/en/java/javase/21/docs/api/java.base/java/util/package-summary.html

### Para os fundamentos da linguagem (Java Language Specification):
https://docs.oracle.com/javase/specs/

## Dica de navegação no site da Oracle:
Quando estiver na página da API, no canto esquerdo você encontra a lista de todos os pacotes. 
É só procurar java.util e clicar para ver todas as classes disponíveis, como Scanner, ArrayList, HashMap, etc.
