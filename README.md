# Java Fundamentos – SENAI

Este repositório contém os materiais, exemplos e exercícios utilizados nas aulas de **Java – Fundamentos**.

## Organização do conteúdo
- Os materiais estão organizados por **módulos**
- Cada módulo possui exemplos práticos em Java
- Utilize este repositório como **material de apoio às aulas**

## Orientações
- Leia os arquivos `README.md` de cada pasta
- Analise os exemplos de código
- Execute os códigos na IDE
